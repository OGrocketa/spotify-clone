- Backend setup:
  1)cd app
  2)pip install -r requirements.txt
  3)uvicorn main:app --reload
  4)The API will be accessible at: http://127.0.0.1:8000

- Frontend setup:
  1)cd frontend
  2)npm install
  3)npm run dev
  4)The frontend will be accessible at: http://localhost:3000.
